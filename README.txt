##Steps to run:
1. Open terminal in folder
2. Run the following commands:
3. make clean
4. make CruiseControl.xes
5. ./CruiseControl.xes
6. The first tick will still be in the OFF state (output CruiseState 1)
7. After the first tick you can change the inputs to try and find alternative outputs

##NOTE:
The Brake and Accel toggles cannot be activated at the same time, this is not expected output
